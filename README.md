# AI_Prompt_Injection_Suite
A tool to help you generate PDFs to be used for prompt injection in AI chatbots. 

## Key features:

4 document templates (invoice, resume, report, contract)
4 injection payload types (increasingly sophisticated)
5 obfuscation techniques (can combine multiple)
Clear feedback on what's happening

## To use:

Pick a template (or edit the visible text)
Pick a payload (or write your own)
Select one or more techniques
Click Generate
